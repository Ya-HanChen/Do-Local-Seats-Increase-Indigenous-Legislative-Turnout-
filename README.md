# Replication Code
This repository contains the R replication code for the analysis in the manuscript titled "Do Local Seats Increase Indigenous Legislative
Turnout? Evidence from the Plain Council Representation in Taiwan."
The code reproduces all main figures and tables reported in the manuscript using regression discontinuity designs.

## Software Requirements
All analyses are conducted using R.

Required user-written R packages:

- rdrobust
- rddensity
- RDDtools

## Data Availability
The dataset used in this study is not publicly available.
Users who have access to the data should place the data file in the data/ directory and ensure that file paths in the .R files are correctly specified.
